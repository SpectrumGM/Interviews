# Prompt Engineering Interview — 9 Core Concepts


| # | Concept | Technique | Best For |
|---|---------|-----------|----------|
| 1 | Zero-Shot & Few-Shot | No examples vs 1–5 examples in prompt | Quick tasks |
| 2 | Chain-of-Thought (CoT) | "Think step-by-step" forcing | Math / logic |
| 3 | Tree-of-Thoughts (ToT) | Multiple reasoning branches, graph reasoning | Planning |
| 4 | ReAct Prompting | Reason + Act loop in one prompt | Tool-using tasks |
| 5 | System & Role Prompting | "You are an expert…" + kitchen rules | Chatbots |
| 6 | Prompt Chaining & Pipelines | Break big task into sequential prompts | Complex workflows |
| 7 | Prompt Optimization | Compression, auto-optimization tools | Production |
| 8 | Evaluation of Prompts | LLM-as-judge, A/B testing | Iteration |
| 9 | Defensive Prompting | Guardrails inside the prompt | High-stakes use |
